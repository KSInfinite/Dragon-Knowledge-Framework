Dragon-Knowledge-Docs ​

Documentation pour Dragon knownledge framework.

Installation ​

Pour installer la documentation, vous pouvez suivres les étapes suivante

  1. Cloner le repository
sh
git clone https://github.com/KSInfinite/Dragon-Knowledge-Docs.git
  1. installation vitepress
sh
npm add -D vitepress
  1. build documentation
sh
npm run docs:build

Change and update doc ​

  1. Faire les mêmes actions que pour l'installation de l'étape 1 à 3
  2. Lancer le mod édition
sh
npm run docs:dev
