SEO ​

Vous pouvez configuez dans le fichier la partie SEO du site dans le fichiers YAML.

It's important to note that the URLs for favicon and sitemap always start in /public/ (but will not be indicated in the path).

Labs ​

Afin de configurer votre zone de test, une section existe également afin de configurer l'accès à ce dernier.

yaml
# Labs/test IP authorization Configuration
labs:
  enable: false
  labsIPAuth:
    - 127.0.0.1

Par défaut l'accès à la section de test est désactivé. Si vous souhaitez autoriser l'accès, il suffit de changer la valeur de enable de false par true comme ci-dessous

yaml
labs:
  enable: true

Par défaut, l'accès à la zone de test n'est autorisé que par l'addresse 127.0.0.1. Si vous souhaitez ajouter d'autre IP ou simplement la remplacer par une autre, il vous suffit de rajouter en dessous d'autre IP comme par exemple (pensez à mettre en true la valeur de enable.) :

yaml
labs:
  enable: true
  labsIPAuth:
    - 127.0.0.1
    - 192.168.1.47
    - 203.0.113.42
