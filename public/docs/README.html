KerogsPHP-Docs ​

Documentation for the KerogsPHP framework.

Installation ​

To install the documentation, follow these steps

  1. Cloning the repository
sh
git clone https://github.com/KSInfinite/KerogsPHP-Docs.git
  1. install vitepress
sh
npm add -D vitepress
  1. build documentation
sh
npm run docs:build

Change and update doc ​

  1. Perform the same actions as for installation steps 1 to 3
  2. Launch edition mod
sh
npm run docs:dev
